# Setup Checklist

Use this checklist to track your progress in customizing your academic website.

## Initial Setup

- [ ] Create repository from template
- [ ] Enable GitHub Pages in repository settings
- [ ] Choose a repository name (e.g., `username.github.io` or project name)

## Configuration (`_config.yml`)

- [ ] Update `title` with your name
- [ ] Update `email` with your email address
- [ ] Update `description` with your bio/research summary
- [ ] Update `author.name` with your full name
- [ ] Update `author.bio` with your title and affiliation
- [ ] Update `author.location` with your city/country
- [ ] Update author links (Email, Google Scholar, GitHub, ORCID, etc.)
- [ ] Update footer links
- [ ] (Optional) Change `minimal_mistakes_skin` to preferred theme

## Profile Photo

- [ ] Replace `assets/images/bio-photo.jpg` with your photo
- [ ] Ensure photo is square and at least 200x200px

## About Page (`_pages/about.md`)

- [ ] Write your biography
- [ ] List your research interests
- [ ] Add recent news/updates
- [ ] Update contact information (email, office, address)

## Research Page (`_pages/research.md`)

- [ ] Add research overview
- [ ] Describe current projects
- [ ] (Optional) Add past projects
- [ ] (Optional) List research group members
- [ ] (Optional) Add information for prospective students

## Publications Page (`_pages/publications.md`)

Choose one approach:

### Option A: Using Data File (Recommended)
- [ ] Edit `_data/publications.yml`
- [ ] Add all your publications
- [ ] Include PDF/DOI/Code links where available

### Option B: Manual Editing
- [ ] Edit `_pages/publications.md` directly
- [ ] Add publications by year
- [ ] Format consistently

## Teaching Page (`_pages/teaching.md`)

- [ ] List current courses
- [ ] List past courses
- [ ] (Optional) Add teaching philosophy
- [ ] (Optional) List students you've advised
- [ ] (Optional) Add teaching awards

## CV Page (`_pages/cv.md`)

- [ ] Update education section
- [ ] Update academic appointments
- [ ] Add publications (or link to publications page)
- [ ] Add grants and awards
- [ ] Add teaching experience
- [ ] Add service activities
- [ ] Add invited talks
- [ ] List students advised
- [ ] Add skills and languages
- [ ] Create PDF version of CV
- [ ] Upload PDF to `assets/files/cv.pdf`

## Navigation (`_data/navigation.yml`)

- [ ] Review navigation items
- [ ] Add/remove/reorder as needed
- [ ] Ensure all links work

## Blog (Optional)

- [ ] Delete old example posts from `_posts/` if not needed
- [ ] Write new blog posts in format `YYYY-MM-DD-title.md`
- [ ] Add proper front matter to posts
- [ ] Decide if you want to keep the blog section

## Final Steps

- [ ] Review all pages on the live site
- [ ] Test all links
- [ ] Check mobile responsiveness
- [ ] Proofread all content
- [ ] (Optional) Set up custom domain
- [ ] Share your website URL!

## Optional Enhancements

- [ ] Add a Publications data file for automatic formatting
- [ ] Create a Talks/Presentations page
- [ ] Add a News/Updates section
- [ ] Include photos of research/lab
- [ ] Add course websites or materials
- [ ] Set up Google Analytics
- [ ] Add social media meta tags
- [ ] Create a sitemap
